Understanding CSS List Properties

list-style-type

Defines the type of marker for list items. Controls the visual style of list bullets or numbers.

Common values: disc, circle, square, decimal, lower-alpha, upper-alpha, upper-roman, none

list-style-position

Specifies whether list item markers appear inside or outside the content flow.

Values: inside (marker inside content), outside (marker outside content)

list-style-image

Replaces the default marker with a custom image. Can be any valid image URL or data URI.

Values: URL, none

list-style (Shorthand)

Combines all list properties in one declaration for cleaner code.

Syntax: list-style: type position image;

Example: list-style: square inside url('bullet.png');
